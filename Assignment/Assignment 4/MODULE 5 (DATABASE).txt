MODULE 5 – DATABASE (Solutions)

1. What is Database?
A Database is an organized collection of data stored in a structured format.

2. What is Normalization?
Normalization is the process of organizing data to reduce redundancy.
Types: 1NF, 2NF, 3NF

3. Difference between DBMS and RDBMS
DBMS: File-based system
RDBMS: Table-based system with relationships

4. Codd Rules
Rules defined by E.F. Codd for RDBMS systems.

5. Data Redundancy
Duplication of data in database.

6. DDL Interpreter
Handles CREATE, ALTER, DROP commands.

7. DML Compiler
Handles SELECT, INSERT, UPDATE, DELETE.

8. SQL Key Constraints Example
CREATE TABLE Student (
    ID INT PRIMARY KEY,
    Name VARCHAR(50),
    Age INT
);

9. Savepoint Example
SAVEPOINT sp1;
ROLLBACK TO sp1;

10. Trigger Example
CREATE TRIGGER after_insert_student
AFTER INSERT ON Student
FOR EACH ROW
INSERT INTO LogTable VALUES (NEW.ID, 'Inserted');

SQL TASKS

Create Student & Exam Tables:
CREATE TABLE Student (
    Student_ID INT PRIMARY KEY,
    Name VARCHAR(50)
);

CREATE TABLE Exam (
    Exam_ID INT PRIMARY KEY,
    Student_ID INT,
    Marks INT
);

Employee Table:
CREATE TABLE Employee (
    Employee_ID INT PRIMARY KEY,
    First_Name VARCHAR(50),
    Joining_Date DATE,
    Salary INT,
    Department VARCHAR(50)
);

Incentive Table:
CREATE TABLE Incentive (
    Incentive_ID INT,
    Employee_ID INT,
    Incentive_Amount INT
);

Queries:
SELECT First_Name AS "Employee Name" FROM Employee;
SELECT First_Name, Joining_Date, Salary FROM Employee;
SELECT * FROM Employee ORDER BY First_Name ASC, Salary DESC;
SELECT * FROM Employee WHERE First_Name LIKE '%J%';
SELECT Department, MAX(Salary) FROM Employee GROUP BY Department;
SELECT e.First_Name, i.Incentive_Amount
FROM Employee e
JOIN Incentive i ON e.Employee_ID = i.Employee_ID
WHERE i.Incentive_Amount > 3000;

Trigger:
CREATE TRIGGER after_employee_insert
AFTER INSERT ON Employee
FOR EACH ROW
INSERT INTO Employee_Log VALUES (NEW.Employee_ID, 'Inserted');

Salesperson Table:
CREATE TABLE Salesperson (
    S_ID INT,
    Name VARCHAR(50),
    City VARCHAR(50),
    Commission FLOAT
);

Customer Table:
CREATE TABLE Customer (
    C_ID INT,
    Name VARCHAR(50),
    City VARCHAR(50),
    Rating INT,
    S_ID INT
);

Queries:
SELECT * FROM Orders WHERE Amount > 1000;
SELECT Name, City FROM Salesperson WHERE City='London' AND Commission>0.12;
SELECT * FROM Salesperson WHERE City IN ('Barcelona','London');
SELECT * FROM Salesperson WHERE Commission > 0.10 AND Commission < 0.12;
SELECT * FROM Customer WHERE Rating > 100 OR City='Rome';
